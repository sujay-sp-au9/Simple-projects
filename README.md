## Caesars-cipher::---
In a shift cipher the meanings of the letters are shifted by some set amount.A common modern use is the ROT13 cipher, where the values of the letters are shifted by 13 places. Thus 'A' ↔ 'N', 'B' ↔ 'O' and so on.
